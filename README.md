# Project Final – Sistema de Reservas para Espacios Comunitarios

Objetivo
Aplicación web para que usuarios puedan reservar espacios comunitarios (salones, canchas, auditorios) con notificación de confirmación. Administradores pueden gestionar y visualizar reservas.

 Tecnologías
- .NET 8 (ASP.NET Core)
- Arquitectura basada en Onion
- PostgreSQL
- Git + GitHub

 Estructura del Proyecto

- `Domain`: Entidades del dominio
- `Application`: Casos de uso
- `Infrastructure`: Repositorios, acceso a datos
- `WebApi`: Controladores HTTP y configuración

 Estado del Proyecto
🚧 En desarrollo (etapa inicial)
